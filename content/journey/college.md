## 大学四年的技术成长

从大一第一次接触 C 语言的迷茫，到大四能够独立完成全栈项目的自信，这四年见证了一个程序员的诞生。

### 学习路径

1. **大一**：C 语言 → 数据结构 → 面向对象编程
2. **大二**：算法竞赛训练 → Java/ Python → 数据库
3. **大三**：Web 开发 → React 生态 → 操作系统/网络
4. **大四**：实习经验 → 毕设项目 → 职业规划

### 竞赛经历

ACM 集训队的训练是大学最难忘的经历之一。每周至少 3 次训练赛，刷了上千道题，从一个只能做水题的小白成长为能够独当一面的队员。
